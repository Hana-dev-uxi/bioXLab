# bioXLab
BioXLab is an interactive biology learning platform designed to make complex biological concepts easier to unserstand through animations, simulations and contextual AI explanations
instead of readi,g static diagrams, users can explore biological processes visually and interactively.

#FEATURES
- interactive biology simulations
- animated educational content
- AI assistant that explains concepts in context
- Disease and biological fact explorer
- "what could go wrong" sections for real-world scenarios
- Responsive UI for desktop

#TECH-STACK
  - HTML
  - CSS
  - JavaScript
  - CloudFlare workers
  - Grok API

#PURPOSE 
BioXLab aims to make biology more interactive, intuitive, and engaging for students and anyone curious about life sciences.
